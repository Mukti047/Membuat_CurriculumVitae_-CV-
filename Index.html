Pekerjaan

Full Stack Web Developer (2025 - Sekarang)
Full Stack Web Developer adalah seseorang yang nyaman bekerja atau merangkap tugas sebagai back end dan front end developer. Lebih spesifiknya, bisa bekerja dengan javascript, PHP, java, database (backend) dan juga bisa mengkonversi desain ke dalam kode pemrograman seperti HTML, CSS, XML (front end).
